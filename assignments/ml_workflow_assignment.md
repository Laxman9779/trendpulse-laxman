# Task 1

Label Column:
The label column is `repeat_purchase_flag` because it represents whether the customer makes a repeat purchase within 30 days, which is the target we want to predict.

Data Leakage Column:
The column `discount_used_on_repeat_order` introduces data leakage because it contains information about the repeat purchase itself, which would not be known at prediction time.
